Katharsis

Επαγγελματικός Καθαρισμός EEG Δεδομένων

Αυτόματη αφαίρεση artifacts από EEG σήματα με προηγμένη ICA τεχνολογία


Δωρεάν Λήψη

Ισχυρά Χαρακτηριστικά

🎯

Αυτόματος Εντοπισμός

Αυτόματος εντοπισμός EOG artifacts με πολλαπλές μεθόδους ανάλυσης και στατιστικούς αλγορίθμους.

🔬

ICA Ανάλυση

Προηγμένη Independent Component Analysis για την ανάλυση των EEG σημάτων σε ανεξάρτητες συνιστώσες.

📊

Οπτικοποίηση

Διαδραστική οπτικοποίηση των ICA συνιστωσών με δυνατότητα επιλογής και σύγκρισης αποτελεσμάτων.

⚡

Γρήγορη Επεξεργασία

Γρήγορη επεξεργασία με multithreading για αποφυγή μπλοκαρίσματος του γραφικού περιβάλλοντος.

📁

EDF Support

Πλήρης υποστήριξη αρχείων EDF από συσκευές EEG όπως το Emotiv Insight και άλλες.

📈

Στατιστική Ανάλυση

Λεπτομερή στατιστικά και μετρήσεις αποτελεσματικότητας για την αξιολόγηση του καθαρισμού.

Εικόνες από την Εφαρμογή

Διαδραστικό περιβάλλον επιλογής ICA συνιστωσών με live preview

Τεχνικές Προδιαγραφές

🐍 Γλώσσα Προγραμματισμού

  • Python 3.8+
  • PyQt6 για GUI
  • MNE-Python για EEG ανάλυση
  • Scikit-learn για ICA

📊 Υποστηριζόμενα Formats

  • EDF (European Data Format)
  • Emotiv Insight 2 συμβατό
  • 10-20 electrode system
  • Multi-channel recording

⚙️ Αλγόριθμοι

  • FastICA Algorithm
  • Band-pass φιλτράρισμα 1-40 Hz
  • Στατιστική ανάλυση (κύρτωση, διακύμανση)
  • Αυτόματος εντοπισμός EOG

💻 Συστήματα

  • Windows 10/11
  • macOS 10.15+
  • Linux (Ubuntu, CentOS)
  • Headless support
